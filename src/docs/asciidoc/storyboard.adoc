== 에러 처리 및 상태 코드

API 호출 시 반환되는 주요 상태 코드는 다음과 같습니다:

* **200:** 성공
* **4xx, 5xx:** 클라이언트 또는 서버 오류 (자세한 내용은 개별 스니펫 참고)
